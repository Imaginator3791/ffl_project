Внутренний отступ. Padding

Padding - это внутренний отстут от границ объекта до границ содержимого внутри этого объекта.

Задается ТОЛЬКО ПОЛОЖИТЕЛЬНЫМ ЧИСЛОМ.

Единицы измерения - px / % / em / rem.

При указании padding: в px; Например padding: 100px; В зависимости от размера объекта динамически изменяется размер содержимого, отступ же при этом остается фиксированным - неизменным.

При указании padding: в %; Например padding: 20%; Значение padding в % высчитывается от ширины родителя. В зависимости от размера ширины родителя динамически изменяется размер И содержимого И отступа padding.

Существует несколько вариантов подключения padding

Перечисление в НЕСКОЛЬКО строк свойств внутреннего отступа с указанием верх 20px - право 30px - низ 40px - лево 100px.
Сокращенная запись в ОДНО значение = padding: 50px; - отступ со всех сторон по 50px.
Сокращенная запись в ДВА значения = padding: 30px 20px;- отступ сверху и снизу - 30x / слева и справа по 20px.
Сокращенная запись в ТРИ значения = padding: 60px 150px 40px; - отступ сверху 60рх / слева и справа по 150рх / снизу 40рх.
Сокращенная запись в ЧЕТЫРЕ значения, указанная в одной строке. Очередность указания как на часах 12 часов - верхний / 3 часа - правый / 6 часов - нижний / 9 часов - левый. В примере верхний 30рх / правый 50рх / нижний 40рх / левый 60рх padding: 30px 50px 40px 60px.

Картинка имеет фиксированну ширину 800px. Поскольку ширина фиксированная, при уменьшении размера родителя картинка наползает на правый паддинг.

Важно!!! Левый padding для картинки работает всегда!!! Если размер картинки превышает [размер содержимого - то отчего считается padding] и [применяемых внтуренних отступов], то картинка вначале наползает сверху на правый padding.

Если родитель и вьюпорт и далее уменьшается, то картинка вылезет за его пределы и появится полоса прокрутки снизу.

Внешний отступ. Margin

margin: ; - это внешний отступ, расстояние от границы объекта до другого объекта.

margin: ; - работает ТОЛЬКО для БЛОЧНЫХ объектов.

Значение margin задаются единицами измерения - px / % / em / rem;

Значение margin может быть как положительным, так и отрицательным!

При использовании отрицательного margin текущий объект поднимается вверх и ВСЕ объекты следующие за ним ТАКЖЕ поднимаются вверх. Важно помнить это свойство отрицательного margin.

Используя внешний отступ / margin: 0 auto; / можно отцентрировать объект по горизонтали. При выполнении следующих условий:

  • Объект должен быть блочным
  • Объект должен быть Уже по ширине чем родитель
  • Объекту задано выравнивание = margin: 0 auto;

Используя внешний отступ / margin: 0 0 0 auto; / можно ПРИБИТЬ блок к правой стороне.

СХЛОПЫВАНИЕ margin.

Если margin задан и снизу и сверху, то он накладывается друг на друга или "схлопывается". При этом если верхний margin 100px, а нижний margin 140px, то схлопывание происходит по бОльшему значению и внешний отступ между объектами составит 140px.

Если верхний margin 50px и нижний margin 50px, то в результате схлопывания внешний отступ между объектами составит 50px.

Существует несколько вариантов подключения margin

Перечисление в НЕСКОЛЬКО строк свойств внешнего отступа margin-top - 30px / margin-right - 40px / margin-bottom - 35px / margin-left - 45px.
Сокращенная запись в ОДНО значение = margin: 80px; - отступ со всех сторон по 80px.
Сокращенная запись в ДВА значения = margin: 30px 20px;- отступ сверху и снизу - 30x / слева и справа по 20px.
Сокращенная запись в ТРИ значения = margin: 60px 150px 40px; - отступ сверху 60рх / слева и справа по 150рх / снизу 40рх.
Сокращенная запись в ЧЕТЫРЕ значения, указанная В ОДНОЙ строке. Очередность указания как на часах 12 часов - верхний / 3 часа - правый / 6 часов - нижний / 9 часов - левый. В примере верхний 30рх / правый 50рх / нижний 40рх / левый 60рх margin: 30px 50px 40px 60px.
