Варианты подключения стилей на странице. Приоритет исполнения.

Существует три вида подключения стилей.

1) Подключаемый файл стилей. Самый распространенный способ подключения стилей! В блоке HEAD прописывается ссылка - <link rel="stylesheet" href="css/style.css" >

2) С использованием атрибутов стилей (инлайново). Используется прямо на странице в коде.
Внимание! Стили указанные в данном варианте перебивают стили указанные ранее, имеют Высший приоритет.

Синтаксис пишется следующим образом:
<p style="font-size: 24px; color:green;" >
Lorem ipsum dolor sit amet consectetur adipisicing elit. Laudantium, vel dolorem! Obcaecati earum molestias dolorem non! </p>

В реальности это выглядит вот так: Lorem ipsum dolor sit amet consectetur adipisicing elit. Laudantium, vel dolorem! Obcaecati earum molestias dolorem non!

3) <style></style> В блоке HEAD, с использованием тега <style>...</style>. Например:
<style>
p {
color: red;
font-size: 14px;
}
</style>

Классы. Структура, спецификация, описание.

Классы используются в подключаемом файле стилей style.css. Можно использовать несколько классов в одном элементе. При этом между селекторами классов нужно ставит обязательно пробел.

Имя класса:

  • Пишется латинскими буквами.
  • Пишется маленькими буквами.
  • Если имя класса состоит из двух и более слов, между ними ставится дефис big-block
  • Рекомендуется называть классы по содержимому.

Анатомия селектора:

  • Селектор начинается с точки - .text
  • Обрамляется фигурными скобками.
  • Между фигурными скобками пишутся css-параметры.
  • css-параметрыотделяются точкой с запятой.

Анатомия css-параметра:

  • Содержит к себе css-свойства.
  • css-свойство:значение; - свойство отделено от значения двоеточием, в конце ставится точка с запятой!

Например:
.text {
color: red;
font-size: 14px;
}
