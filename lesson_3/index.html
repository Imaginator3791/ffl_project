Варианты подключения стилей на странице. Приоритет исполнения.

Существует три вида подключения стилей.

1) Подключаемый файл стилей. Самый распространенный способ подключения стилей! В блоке HEAD прописывается ссылка

2) С использованием атрибутов стилей (инлайново). Используется прямо на странице в коде.
Внимание! Стили указанные в данном варианте перебивают стили указанные ранее, имеют Высший приоритет.

В реальности это выглядит вот так: Lorem ipsum dolor sit amet consectetur adipisicing elit. Laudantium, vel dolorem! Obcaecati earum molestias dolorem non!

Классы. Структура, спецификация, описание.

Классы используются в подключаемом файле стилей style.css. Можно использовать несколько классов в одном элементе. При этом между селекторами классов нужно ставит обязательно пробел.

Имя класса:

  • Пишется латинскими буквами.
  • Пишется маленькими буквами.
  • Если имя класса состоит из двух и более слов, между ними ставится дефис big-block
  • Рекомендуется называть классы по содержимому.

Анатомия селекторов:

Селектор - это механизм, с помощью которого мы обращаемся к объектам внутри HTML-файла. Селекторов существует нескольк типов.

  • Обрамляется фигурными скобками.
  • Между фигурными скобками пишутся css-параметры.
  • css-параметры отделяются точкой с запятой.

Анатомия css-параметра:

  • Сss-параметр содержит в себе css-свойства и значения.
  • css-свойство:значение - свойство отделено от значения двоеточием, в конце ставится точка с запятой!

Селекторы. Типы. Приоритетность.

  • Селектор ID
  • Селектор класса
  • Селектор тега

Селектор ID

Приоритет ОДИН - Селектор ID приоритетнее / выше, чем селектор класса. Имеет высший приоритет - БОЖЕСТВО )))

Селектор ID - это обращение к ID объекта.
Идентификатор может быть только один на странице.
Также нельзя в свойствах объекта нельзя указывать несколько идентификаторов.

Пример: Здесь используется селектор ID, который раскрасит в пурпурный цвет цвет и размер шрифта 16хр Lorem ipsum dolor sit amet consectetur.

Селектор класса.

Приоритет ДВА - Селектор класса приоритетнее / выше, чем селектор тега.

Селектор класса начинается с точки и далее в фигурных скобках перечисляются свойства и значения класса.
Селектор класса уточняет какому элементу страницы присвоить нужные свойства и параметры. То есть не всем тегам span на странице, а только тем, которым присвоен класс.
Класс может быть присвоен любому объекту на странице

Классов у одного объекта может быть много. В стилях одного объекта несколько классов. Отделяются друг от друга пробелом.

Пример: Здесь используется селектор класса, который раскрасит в синий цвет и размер шрифта 18хр Lorem ipsum dolor sit amet consectetur.

Селектор тега.

Приоритет ТРИ - Селектор тега имеет наименьший приоритет среди селекторов.

Например тег span. При формировании стиля таким образом указанные свойства будут присвоены ВСЕМ тегам span на странице.

Примеp: Здесь в коде указан тег span Lorem ipsum dolor sit amet consectetur.

Уточнение обращения.

Возможны несколько вариантов обращений к элементам страницы. Возможно написание класса стиля который может быть применен где угодно. Например .text{} c соответствующими свойствами и параметрами.
А возможно уточнение div.text{} с соответствующими свойствами и парметрами, который будет применен ТОЛЬКО к элементам DIV.

Пример: Мы имеем абзац P c классом .text цвет текста ЗЕЛЕНЫЙ и имеем уточнение div.text цвет текста индиго

Комбинация селекторов класса.

Возможно уточнение по комбинации селекторов класса. Например в одном объекте присутствует класс .text, а в другом присутствуют два класса - например .text1 и .prargraph.
Если указать класс .text1.paragraph{} то такой класс будет искать объекты где присутствуют оба класса ОДНОВРЕМЕННО и будет иметь больший приоритет, чем свойства объекта где указан только класс .text1{}.
Чем больше в ряду классов, присутствующих одновременно, тем запись приоритетнее.

Пример: Мы имеем абзац P c классом .text1.paragraph цвет текста СИНИЙ и имеем DIV c классом .text1 - цвет текста ЗЕЛЕНЫЙ

Пример: Мы имеем абзац P c классом .text1.paragraph цвет текста СИНИЙ и имеем DIV c классом .text1 - цвет текста ЗЕЛЕНЫЙ

Вложенность css-селекторов.

Вложенность стилей позволяет обращаться к элементу внутри объекта.
Например: У нас есть элемент с тегом SPAN внутри объекта Р. Соответственно если у SPAN на странице вообще стиль цвета текста зеленый, то SPAN внутри можно раскрасить в например в красный цвет.
Чтобы выделить SPAN внтури объекта P пишется следующий селектор p span {} - p(пробел)span {}

Селекторы атрибутов.

Существуют разные механизм обращения к селекторам атрибутов классов

  • Перечисление
  • Использование селектора атрибута
  • Комплексные селекторы

Перечисление

Например, на странице есть несколько объектов about__text, block__text.
В файле стилей можно через запяткю перечислить эти селекторы и присвоить им необходимые свойства. Мы присвоим в файле стилей данным селекторам розовый цвет. .about__text,.block__text{...}

Использование селектора атрибута

При использовании селектора атрибута пишется регулярное выражение.
В квадратнх скобках [class*="общая часть селектора"] {...}
Данное выражение ищет на странице общую часть селектора и присваивает уазанные в файле css свойства.
* в данном случае означает "содержит"
